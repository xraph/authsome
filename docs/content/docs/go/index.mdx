**AuthSome** is a comprehensive, pluggable authentication framework for Go, inspired by better-auth (TypeScript). It provides enterprise-grade authentication with multi-tenancy support, designed to be mounted on the [Forge framework](https://github.com/xraph/forge).

<Callout type="info">
AuthSome Go is currently in active development. Check our [GitHub repository](https://github.com/xraph/authsome) for the latest updates and contribute to the project.
</Callout>

## Architecture Overview

AuthSome follows clean architecture principles with clear separation of concerns:

<Tabs items={['Standalone Mode', 'SaaS Mode']}>
  <Tab value="Standalone Mode">
    Perfect for single-tenant applications:
    - Single default organization (auto-created)
    - Organization creation APIs disabled
    - No org-scoped configs
    - Simple single-tenant setup
  </Tab>
  <Tab value="SaaS Mode">
    Full multi-tenancy support:
    - Multiple isolated organizations
    - Organization creation enabled
    - Org-scoped config overrides
    - Platform organization for system-wide settings
  </Tab>
</Tabs>

## Technology Stack

- **Framework**: [Forge](https://github.com/xraph/forge) - Modern Go web framework
- **ORM**: [Bun](https://bun.uptrace.dev/) - Fast and simple Go ORM
- **Forms**: [Smartform](https://github.com/xraph/smartform) - Type-safe form handling
- **Storage**: Redis for session caching and rate limiting
- **Database**: PostgreSQL, MySQL, SQLite support

## Quick Start

Get started with AuthSome in minutes:

<Callout type="tip">
Make sure you have Go 1.21+ installed and a supported database running.
</Callout>

```bash
# Install AuthSome
go get github.com/xraph/authsome

# Create a new project
mkdir my-auth-app && cd my-auth-app
go mod init my-auth-app

# Initialize with basic configuration
authsome init --mode=standalone
```

```go
package main

import (
    "github.com/xraph/authsome"
    "github.com/xraph/forge"
)

func main() {
    // Create Forge app
    app := forge.New()
    
    // Initialize AuthSome
    auth, err := authsome.New(authsome.Config{
        Mode: authsome.Standalone,
        Database: authsome.DatabaseConfig{
            Driver: "postgres",
            URL:    "postgres://user:pass@localhost/mydb",
        },
    })
    if err != nil {
        panic(err)
    }
    
    // Mount AuthSome to your app
    auth.Mount(app, "/auth")
    
    // Start server
    app.Listen(":8080")
}
```
