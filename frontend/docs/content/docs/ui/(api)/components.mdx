# Components Reference

This page documents all components from `@authsome/ui-react`, `@authsome/ui-react-headless`, and `@authsome/ui-next`.

## @authsome/ui-react

### AuthProvider

The root context provider. Wrap your app with this component.

```typescript
interface AuthProviderProps {
  /** AuthClient instance (required) */
  client: AuthClient;
  children: React.ReactNode;
  /** Flow engine configuration */
  flows?: FlowConfig;
  initialFlowState?: Partial<FlowState>;
  /** Override default UI components used by flow renderers */
  uiComponents?: Partial<UIComponents>;
  /** Renderer configuration for built-in flows */
  rendererConfig?: RendererConfig;
  /** Flow state change callback */
  onFlowStateChange?: (state: FlowState) => void;
  /** Locale overrides for i18n */
  locale?: DeepPartial<AuthLocale>;
  /** Organization change callback */
  onOrganizationChange?: (org: Organization | null) => void;
}
```

### RequireAuth

Renders children only when authenticated. Shows fallback or redirects otherwise.

```typescript
interface RequireAuthProps {
  children: React.ReactNode;
  /** Shown while loading or when unauthenticated */
  fallback?: React.ReactNode;
  /** URL to redirect to when unauthenticated */
  redirectTo?: string;
  /** Callback when user is not authenticated */
  onUnauthenticated?: () => void;
}
```

### withAuth

HOC that wraps a component with auth protection.

```typescript
interface WithAuthOptions {
  fallback?: React.ComponentType;
  redirectTo?: string;
  onUnauthenticated?: () => void;
}

function withAuth<P extends object>(
  Component: React.ComponentType<P>,
  options?: WithAuthOptions
): React.ComponentType<P>
```

