## Quick Setup

<Tabs items={['YAML Configuration', 'Programmatic']}>
  <Tab value="YAML Configuration">
    ```yaml
    auth:
      plugins:
        # Passkey authentication
        passkey:
          rpId: "example.com"
          rpName: "Example App"
          rpOrigins:
            - "https://example.com"
          timeout: 60000
          userVerification: "preferred"

        # Magic link authentication
        magicLink:
          expiryMinutes: 15
          baseURL: "https://example.com"
          allowImplicitSignup: true
          rateLimitPerHour: 10

        # Social OAuth
        social:
          autoCreateUser: true
          allowAccountLinking: true
          trustEmailVerified: true
          providers:
            google:
              clientId: "${GOOGLE_CLIENT_ID}"
              clientSecret: "${GOOGLE_CLIENT_SECRET}"
            github:
              clientId: "${GITHUB_CLIENT_ID}"
              clientSecret: "${GITHUB_CLIENT_SECRET}"

        # JWT tokens
        jwt:
          issuer: "authsome"
          accessExpirySeconds: 3600
          refreshExpirySeconds: 2592000
          signingAlgorithm: "HS256"

        # API keys
        apikey:
          defaultRateLimit: 1000
          maxKeysPerUser: 10
          keyLength: 32
          autoCleanup:
            enabled: true
            interval: "24h"

        # Multi-session
        multisession:
          maxSessionsPerUser: 10
          enableDeviceTracking: true
          sessionExpiryHours: 720
    ```
  </Tab>
  <Tab value="Programmatic">
    ```go
    import (
        "github.com/xraph/authsome"
        "github.com/xraph/authsome/plugins/passkey"
        "github.com/xraph/authsome/plugins/magiclink"
        "github.com/xraph/authsome/plugins/social"
        "github.com/xraph/authsome/plugins/jwt"
        "github.com/xraph/authsome/plugins/apikey"
        "github.com/xraph/authsome/plugins/multisession"
        "github.com/xraph/authsome/plugins/notification"
    )

    func setupAdvancedPlugins(auth *authsome.Auth) error {
        auth.RegisterPlugin(passkey.NewPlugin())
        auth.RegisterPlugin(magiclink.NewPlugin())
        auth.RegisterPlugin(social.NewPlugin())
        auth.RegisterPlugin(jwt.NewPlugin())
        auth.RegisterPlugin(apikey.NewPlugin())
        auth.RegisterPlugin(multisession.NewPlugin())
        auth.RegisterPlugin(notification.NewPlugin())

        return auth.InitializePlugins()
    }
    ```
  </Tab>
</Tabs>

## Common API Patterns

### Passkey Plugin
```http
POST /passkey/register/begin      # Begin passkey registration
POST /passkey/register/finish     # Complete passkey registration
POST /passkey/login/begin         # Begin passkey login
POST /passkey/login/finish        # Complete passkey login
GET  /passkey/list                # List registered passkeys
```

### Social OAuth Plugin
```http
POST /signin/social               # Initiate social sign-in
GET  /callback/:provider          # OAuth callback handler
POST /account/link                # Link social account
POST /account/unlink/:provider    # Unlink social account
GET  /providers                   # List available providers
```

### JWT Plugin
```http
POST /jwt/keys                    # Create JWT signing key
GET  /jwt/keys                    # List JWT keys
POST /jwt/generate                # Generate JWT token
POST /jwt/verify                  # Verify JWT token
GET  /jwt/jwks                    # JWKS endpoint (RFC 7517)
```

### API Key Plugin
```http
POST /api-keys                    # Create API key
GET  /api-keys                    # List API keys
GET  /api-keys/:id                # Get API key details
PUT  /api-keys/:id                # Update API key
DELETE /api-keys/:id              # Delete API key
POST /api-keys/:id/rotate         # Rotate API key
POST /api-keys/verify             # Verify API key
```

### Multi-Session Plugin
```http
GET  /multi-session/list          # List all sessions
POST /multi-session/set-active    # Set active session
DELETE /multi-session/delete      # Delete session
GET  /multi-session/current       # Get current session
POST /multi-session/revoke-all    # Revoke all sessions
POST /multi-session/revoke-others # Revoke other sessions
POST /multi-session/refresh       # Refresh session
GET  /multi-session/stats         # Session statistics
```

### Notification Plugin
```http
POST /templates                   # Create notification template
GET  /templates                   # List templates
POST /notifications/send          # Send notification
POST /notifications/webhook/:provider  # Provider webhook
```

## Security Considerations

<Callout type="warn">
**Security Best Practices**: Advanced plugins require careful security configuration, especially for passwordless authentication methods and token management.
</Callout>

### WebAuthn Security (Passkey)
- Configure `rpId` to match your domain exactly
- Use HTTPS in production (required by WebAuthn spec)
- Set `userVerification` to "required" for sensitive operations

### Social OAuth Security
- Store client secrets in environment variables, never in code
- Use state parameter for CSRF protection (handled automatically)
- Configure allowed redirect URLs precisely

### JWT Security
- Rotate signing keys regularly
- Use appropriate algorithm (RS256 for distributed systems, HS256 for single service)
- Set reasonable token expiry times

### API Key Security
- Enable rate limiting per key
- Use IP whitelisting for server-to-server keys
- Enable automatic cleanup of expired keys
- Rotate keys periodically
